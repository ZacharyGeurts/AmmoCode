Programmerland

A design theme: tools and docs that are kind to programmers and to AI agents working beside them.

For programmers

  • Plain words — errors and docs say what happened and what to try next.
  • One compile path — g16 discern, AmmoCode Run/Build, no mystery chains.
  • Non-destructive editor — read-only disk API; save means export to a path you choose.
  • 268 file extensions — DOS, BBS, legacy BASIC, modern Rust/TS — same respectful open flow.
  • Loopback only — 127.0.0.1; your code stays on your machine.

For AI agents

  • AI primers per language tab — copy-paste brain-sync without inventing APIs.
  • Grok16 MCP — grok16_version, toolchain, rtx_gate with stable JSON.
  • Truthful context — agents are told when APIs are read-only or export-only.
  • Single language registry — grok16-languages.json (79 entries); no parallel fiction.
