# AmmoCode Wiki

**Live editor:** [zacharygeurts.github.io/AmmoCode](https://zacharygeurts.github.io/AmmoCode/)  
**Repo:** [ZacharyGeurts/AmmoCode](https://github.com/ZacharyGeurts/AmmoCode)

AmmoCode is the **NEXUS C2 stack editor** — lean, non-destructive, 268 text-era filetypes, paired with **Grok16** for local compile.

## Best version

| Piece | Source of truth | Version |
|-------|-----------------|---------|
| Editor + GitHub Pages | This repo (`AmmoCode`) | **6.1.0** |
| Compiler `g16` / `g++16` | **Grok16** hard plane (Field / AmmoOS) | **16.1.0-hard**+ |
| Full OS stack | [AmmoOS](https://github.com/ZacharyGeurts/AmmoOS) | beta / release tags |

When both SG `AmmoOS/AmmoCode` and Grok16 are available: **ship the AmmoCode 6.1 UI**, **pair Grok16 for compile**. Pages never needs Python.
