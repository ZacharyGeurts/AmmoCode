# Filetypes

Canonical DB: `data/field-programming-filetypes.json` (**268** extensions).

## Eras

- **Modern** — TS, Python, Rust, C/C++, Markdown, …
- **DOS / BBS** — `.nfo`, `.ans`, CONFIG.SYS, …
- **Legacy languages** — BASIC, Fortran, COBOL, Pascal, …
- **Plain text** — logs, lists, TeX

## Binary guard

Executables, images, and archives are rejected unless the extension is registered as a programming text type.

## Discern

Local g16: `g16 --g16-discern` (when loopback is up).  
Pages: extension map + client highlight only.
