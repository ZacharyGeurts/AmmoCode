AmmoCode 2027

THE compiler GUI — sovereign editor on the g16 / Grok16 stack. Zero telemetry. Loopback only.

AI Primer — switch language to load a copy-paste tutorial for your AI. Toolbar copy/paste or tab clipboard icons.

Tab aging — idle tabs warm to red over 3 hours. Right-click a tab to disable color or remember per file.

Security — transparent vuln scan, hardened rewrite, combinatorics-aware blocks before build.

Collab — invite-only WebSocket room, IP friends, voice (mic/volume/mute), cursor personas, text chat.

Network mesh — polite LAN discovery (rate-limited, not spammy). Friend/block lists. Automated threat ratings. HTTP tunnel connects local + open peers when WebSocket is blocked.

Host evaluation — on open, AmmoCode scores this machine using connection gatekeeper + ZNetwork hostile scan + local heuristics.

ZNetwork — NewLatest field secure network hooks through when AmmoCode opens. If ZNetwork is already running, AmmoCode attaches only (no interference). Hook depth: editor → g16 security → DDoS guard → ZNetwork field bridges.

Shield — no screen capture or keyloggers unless host permits collab GUI share. File → Screenshot… always available. MITM beacon pins block tunnel hijacks.

Defield — if AmmoCode or SG/Grok16 is fielded, auto-defield on boot (no field-on-field).

Run — g16 syntax check and build with belt profiles. Activity bar flyouts for collab, security, combinatorics, ZNetwork.

Shortcuts: Ctrl+S save · Esc close flyout
